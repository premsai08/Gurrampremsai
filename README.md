# G Prem Sai Portfolio

Executive portfolio for G Prem Sai, built with React, Vite, Framer Motion, Tailwind CSS, and a custom Three.js hero scene.

## Local Setup

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

Vercel redeploys automatically when changes are pushed to the connected `main` branch.
